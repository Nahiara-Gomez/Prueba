Descripcion del proyecto:
Este proyecto consiste en el diseño e implementación de una aplicación de escritorio para registrar y publicar inasistencias docentes,
especialmente aquellas vinculadas a licencias médicas. La aplicación esta desarrollada en Java con orientación a objetos, 
cuenta con una interfaz construida en Swing (NetBeans) y gestiona los datos mediante una base de datos relacional (MySQL).
La información visible al público incluye el nombre y apellido del docente, turno, grupos afectados, fecha de incio y finalización de la licencia
y motivo de la inasistencia.
Solamente los administradores que esten registrados en la base de datos podran ingresar y eliminar inasistencias.
Y los alumnos podran filtrar que inasistencias desean ver, segun el turno al que asisten.

Tecnologias usadas: gestor de base de datos (Mysql) y Neats Beans 

Que se precisa para probarlo: para poder probar adecuadamente la aplicacion, se precisa tener instalado Neat beans, el conector de base de datos, Xammp (para poder iniciar
la base de datos mysql) y los archivos .zip y .sql de nuestro programa. En cuanto a ram se precisa tener..
